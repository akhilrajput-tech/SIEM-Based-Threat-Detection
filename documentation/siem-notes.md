# SIEM-Based Threat Detection Notes

## What is SIEM?
SIEM (Security Information and Event Management) collects, analyzes, and correlates logs to detect security threats.

## Key SIEM Components
- Log collection
- Normalization
- Correlation rules
- Alerts and dashboards

## SOC Analyst Role
- Monitor alerts
- Analyze logs
- Identify false positives
- Escalate real incidents

## Best Practices
- Use meaningful detection rules
- Monitor high-risk events
- Regularly tune alerts
